## Next Steps to Push to GitHub

### ⭐ Recommended: Use SSH

```bash
# 1. Check SSH key
ls ~/.ssh/id_rsa

# 2. If needed, generate SSH key
ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa -N ""

# 3. Copy public key
cat ~/.ssh/id_rsa.pub

# 4. Add to GitHub: https://github.com/settings/ssh/new

# 5. Change remote to SSH
git remote set-url origin git@github.com:singhatharav935/sannidh-command.git

# 6. Push
cd /Users/atharavsingh/Desktop/SANNIDH_MASTER/frontend
git push origin main
```

### Alternative: Use Personal Access Token (PAT)

```bash
# 1. Create PAT at: https://github.com/settings/tokens/new
#    Scope: repo (full control)
#    Expiry: 90 days

# 2. Store in macOS Keychain
git config --global credential.helper osxkeychain

# 3. Push (will prompt for credentials)
git push origin main
# Username: <your-github-username>
# Password: <your-PAT>
```

### Alternative: Direct Push with PAT

```bash
git push https://<YOUR_PAT>@github.com/singhatharav935/sannidh-command.git main
```
