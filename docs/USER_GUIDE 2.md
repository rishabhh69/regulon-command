# SANNIDH User Guide

## Getting Started (5 minutes)

### 1. Sign Up
1. Go to https://sannidh.ai
2. Click "Register"
3. Enter email, password, full name
4. Select your role (Company Owner, CA, Lawyer, etc.)
5. Click "Register as [Role]"
6. Check email for verification link

### 2. Login
1. Go to https://sannidh.ai/auth
2. Enter email & password
3. Click "Login as [Role]"
4. You're in! 🎉

### 3. Your Dashboard
- **Company Owner**: Compliance overview, deadlines, tasks
- **External CA**: Client management, filings, AI tools
- **CA Firm**: Team management, workload distribution
- **In-House CA/Lawyer**: Company-specific compliance
- **Admin**: Platform administration

## Key Features

### 🤖 AI Compliance Drafting
1. Go to "AI Drafting" tab
2. Select document type (GST Return, Income Tax, etc.)
3. Fill required details
4. Click "Generate Draft"
5. Review and download

### 💬 Compliance Chatbot
1. Click chatbot icon (bottom-right)
2. Ask compliance questions
3. Get instant answers
4. Cite sources automatically

### 📊 Dashboard Analytics
- Compliance score
- Upcoming deadlines
- Task progress
- Risk indicators

### 📁 Document Management
1. Upload documents (drag & drop)
2. Organize by category
3. Search & filter
4. Download anytime

### ✅ Task Management
1. Create tasks
2. Assign deadlines
3. Track progress (Kanban board)
4. Mark complete

## FAQ

**Q: Can I use same email for multiple roles?**
A: Yes! Different passwords required.

**Q: Is my data secure?**
A: Yes. HTTPS, encryption, GDPR compliant.

**Q: How do I export my data?**
A: Account Settings → Export Data

**Q: How do I delete my account?**
A: Account Settings → Delete Account (permanent!)

**Q: Mobile app available?**
A: Responsive web app works on all devices.
