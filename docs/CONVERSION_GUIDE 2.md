### **OPTION 3: Command Line (If you have LaTeX)**

Install LaTeX if needed:
```bash
brew install mactex  # Mac
```

Then convert:
```bash
pandoc /Users/atharavsingh/Desktop/SANNIDH_MASTER/frontend/DEMO_VIDEO_SCRIPT.md \
       -o /Users/atharavsingh/Desktop/SANNIDH_MASTER/frontend/DEMO_VIDEO_SCRIPT.pdf \

```

**Result:** Optimized PDF with full formatting

## 🆘 Troubleshooting

**Browser print not working?**
- Try a different browser (Chrome, Firefox, Safari)
- Clear cache and cookies
- Check if JavaScript is enabled

**HTML not displaying properly?**
- Update your browser
- Try a different browser
- Check file permissions

**Want different styling?**
- Edit the `<style>` section in the HTML file
- Adjust colors, fonts, spacing as needed
- Save and print again
